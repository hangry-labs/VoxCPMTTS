VoxCPMTTS

Easy-to-run Docker packaging for VoxCPM2, a realistic multilingual text-to-speech model with voice design, controllable cloning, and transcript-guided cloning. Run the image, open the browser UI, or call the same HTTP API from your own application.

30 languages 48 kHz output Voice clone Voice design UI + API

Run

docker run -p 8808:8808 --gpus all hangrylabs/voxcpmtts:v0.1

Open

http://localhost:8808
